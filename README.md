# Titanic Survival Prediction 🚢

## Project Overview
Built an ML model to predict whether a Titanic passenger 
survived or not based on their details.

## Results
- Accuracy: 82.12%
- Cross Validation Accuracy: 81.26%
- Algorithm: Random Forest Classifier

## Key Finding
Fare (ticket price) was the most important feature — 
passengers with expensive tickets had higher survival rates
due to better facilities and closer access to lifeboats.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

## Dataset
Titanic dataset — 891 passengers, 12 features

## How to Run
1. Open the .ipynb file in Google Colab
2. Run all cells in order
